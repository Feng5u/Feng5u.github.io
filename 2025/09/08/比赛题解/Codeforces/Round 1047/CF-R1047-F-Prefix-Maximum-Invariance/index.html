[CF R1047] F. Prefix Maximum Invariance

Statement

给定两个数列 {xn}\{x_n\}{xn​} 和 {yn}\{y_n\}{yn​}，定义 f(x,y)f(x,y)f(x,y) 为：根据 xxx 构造一个数列 zzz，使得二者任意一段前缀的最大值相同，f(x,y)f(x,y)f(x,y) 即为 zzz 可能与 yyy 相同的位数的个数。

求 ∑l=1n∑r=lnf([xl,xl+1,…,xr],[yl,yl+1,…,yr])\sum_{l=1}^n \sum_{r=l}^n f([x_l,x_{l+1},\dots,x_r], [y_l,y_{l+1},\dots,y_r])∑l=1n​∑r=ln​f([xl​,xl+1​,…,xr​],[yl​,yl+1​,…,yr​])。

1≤n≤2×105,1≤xi,yi≤2n1 \leq n \leq 2\times 10^5, 1 \leq x_i,y_i \leq 2n1≤n≤2×105,1≤xi​,yi​≤2n

Tutorial

首先考虑，当 x,yx,yx,y 给定的时候，f(x,y)f(x,y)f(x,y) 如何计算。首先，如果 xix_ixi​ 是 x[1…i]x[1\dots i]x[1…i] 的最大值，那么 yiy_iyi​ 必须与 xix_ixi​ 相等，才有可能有 zi=yiz_i = y_izi​=yi​。称这样的 xix_ixi​ 为“重要的”。而一个 xix_ixi​ 不是重要的，当且仅当它小于等于前一个重要的 xix_ixi​，此时只要 yiy_iyi​ 也小于等于前一个重要的 xix_ixi​ 即可。

下面考虑，对于 x,yx,yx,y 的一段前缀，当其长度加一时，会产生什么影响。首先，对于新前缀的各个后缀，新元素的加入并不影响老元素的贡献，因此这部分贡献直接累加即可。而新元素的新贡献可以按照上述情况分类讨论。下面，我们需要确定当前位是否是重要的。显然，如果当前元素在某个后缀中是重要的，它一定是这个后缀的最大值，因此我们只需要找到第一个大于当前元素的元素即可，这可以用单调队列维护。
