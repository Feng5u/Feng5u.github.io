[CF R1049] D. A Cruel Segment's Thesis

Statement

给定 nnn 段未标记的线段，每次选择两个未标记的线段 iii 和 jjj，标记二者，并构造一个新的被标记的线段 [x,y][x,y][x,y]，满足 x∈[li,ri]x\in[l_i,r_i]x∈[li​,ri​]，y∈[lj,rj]y\in[l_j,r_j]y∈[lj​,rj​]，求最后所有被标记的线段的长度总和的最大值。（如果未标记的线段数量为 111，则直接标记）

1≤n≤2×105,1≤li<ri≤1091 \leq n\leq 2\times 10^5, 1 \leq l_i < r_i \leq 10^91≤n≤2×105,1≤li​<ri​≤109

Tutorial

首先考虑 nnn 为偶数时，此时所有初始的线段都可以被用来产生新的线段。实际上相当于最大化：

∑i=1n/2ri−∑j=1n/2lj\sum_{i = 1}^{n / 2} r_i - \sum_{j = 1}^{n / 2} l_j i=1∑n/2​ri​−j=1∑n/2​lj​

如果直接排序，会出现同一个线段的左右端点都被计入的情况，因此进行转化，化为：

∑i=1nri−∑j=1n/2(rj+lj)\sum_{i=1}^nr_i - \sum_{j=1}^{n / 2}(r_j+l_j) i=1∑n​ri​−j=1∑n/2​(rj​+lj​)

于是只需要将 ri+lir_i+l_iri​+li​ 排序即可。

当 nnn 为奇数时，只需要枚举去掉的线段即可。

时间复杂度：O(nlog⁡n)O(n\log n)O(nlogn)
