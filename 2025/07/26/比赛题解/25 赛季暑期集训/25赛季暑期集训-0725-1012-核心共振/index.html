[25赛季暑期集训 0725] 1012 核心共振

Statement

给定 nnn 个核心，每个核心有一个位置 (xi,yi)(x_i, y_i)(xi​,yi​) 和一个参数 aia_iai​，每对核心的共振度定义为 (ai+aj)⋅max⁡(∣xi−xj∣,∣yi−yj∣)(a_i + a_j) \cdot \max(|x_i - x_j|, |y _ i- y_j|)(ai​+aj​)⋅max(∣xi​−xj​∣,∣yi​−yj​∣)，求出所有核心两两的共振度之和。

1≤n≤2×1051 \leq n \leq 2 \times 10^51≤n≤2×105

Tutorial

由数学关系，可以得到：max⁡(∣xi−xj∣,∣yi−yj∣)=∣(xi−yi)−(xj−yj)∣+∣(xi+yi)−(xj+yj)∣2\max(|x_i - x_j|, |y_i-y_j|) = \dfrac{|(x_i - y_i) - (x_j - y_j)| + |(x_i + y_i) - (x_j + y_j)|}{2}max(∣xi​−xj​∣,∣yi​−yj​∣)=2∣(xi​−yi​)−(xj​−yj​)∣+∣(xi​+yi​)−(xj​+yj​)∣​。不妨先考虑 x+yx + yx+y，计算 ∑1≤i≤j≤n(ai+aj)⋅∣(xi+yi)−(xj+yj)∣\sum_{1 \leq i \leq j \leq n}(a_i + a_j) \cdot |(x_i + y_i) - (x_j + y_j)|∑1≤i≤j≤n​(ai​+aj​)⋅∣(xi​+yi​)−(xj​+yj​)∣。 令 w=x+yw = x + yw=x+y，升序排列，得到 ∑1≤i≤j≤n(ai+aj)(wj−wi)\sum_{1 \leq i\leq j\leq n} (a_i + a_j)(w_j - w_i)∑1≤i≤j≤n​(ai​+aj​)(wj​−wi​)，将括号拆开后就变成了对前缀和和后缀和进行的操作。

时间复杂度：O(nlog⁡n)O(n \log n)O(nlogn)，主要来自于排序
