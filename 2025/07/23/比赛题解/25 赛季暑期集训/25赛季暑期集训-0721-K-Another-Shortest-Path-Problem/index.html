[25赛季暑期集训 0721] K. Another Shortest Path Problem

Statement

给定 nnn 个节点和 nnn 条边，进行 qqq 次询问，每次给定两个点 xxx 和 yyy，求两点之间的最短路长度。

1≤n≤105,1≤q≤1051 \leq n \leq 10^5, 1 \leq q \leq 10^51≤n≤105,1≤q≤105

Tutorial

由于有 nnn 条边，因此可以看作是在一棵树上加了一条边，于是图变成了：一个环、环上每个节点都可能有一棵子树。于是可以先找到这个环，处理出环上任意两点的距离（可以通过记录所有点到某一个点的距离，然后查询的时候 O(1)O(1)O(1) 的处理出来距离）。然后对于环上每一个点，处理其子树上任意一点到它的距离，并为倍增做好准备。

接下来，讨论查询的两点的关系：

  1. 所在子树根节点不同。可以 O(1)O(1)O(1) 地得到两根节点距离以及两点到各自根节点的距离；
  2. 所在子树根节点相同。倍增求其 LCA\text{LCA}LCA 即可

时间复杂度： O(n+qlog⁡n)O(n + q\log n)O(n+qlogn)
