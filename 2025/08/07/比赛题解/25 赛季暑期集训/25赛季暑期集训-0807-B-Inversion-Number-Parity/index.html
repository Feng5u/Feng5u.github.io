[25赛季暑期集训 0807] B - Inversion Number Parity

Statement

给定随机数生成器参数，基于交换生成一个 0 到 (n − 1) 的随机排列，并生成 (n − 1) 次区间循环左移的修改，问修改前后每次排列的逆序对奇偶性。

1≤n≤1071 \leq n \leq 10^71≤n≤107

Tutorial

首先考虑区间左移的操作对逆序对的奇偶性有什么影响。区间左移等价于将区间分为两半，两个区间内部的逆序对数不会发生改变。考虑前一个区间中的某个数字，假设它与后一个区间形成了 kkk 个逆序对，后一个区间一共有 nnn 个数字，那么当这个数字移动到后一个区间后面时，有 kkk 个逆序对消失了，产生了 n−kn-kn−k 个逆序对，因此逆序对数改变了 n−2kn-2kn−2k 次，显然，这与 2k2k2k 没什么关系，仅与 nnn 有关。

下面考虑生成数据的过程。由于数据生成器是一个仅与前三项有关的递推式，因此可以使用轮换数组，唯一的难点在于利用交换构造排列。由于空间限制，无法存储排列，但是由于排列是通过交换生成的，可以利用线性代数中的结论，直接得到逆序对个数的奇偶性。

时间复杂度：O(n)O(n)O(n)
