快速沃尔什变换

原理

FWT\text{FWT}FWT 也是加速多项式卷积的工具，但其处理的卷积的形式如下：

ci=∑j⊗k=iajbkc_i = \sum_{j \otimes k = i} a_jb_k ci​=j⊗k=i∑​aj​bk​

其中，⊗\otimes⊗ 表示按位与、按位或、按位异或中的一种。

类似 FFT\text{FFT}FFT，FWT\text{FWT}FWT 也是将两个多项式转换成可以逐项相乘的形式，再通过逆变换进行还原。

按位或

进行如下正变换：

Ai=∑j∣i=iajA_i = \sum_{j | i = i} a_j Ai​=j∣i=i∑​aj​

该变换满足 Ci=AiBiC_i = A_iB_iCi​=Ai​Bi​，且可以基于以下性质快速进行：

A=(A0,A0+A1)A = (A_0,A_0+A_1) A=(A0​,A0​+A1​)

其中，A0A_0A0​ 表示对 aaa 的前半部分做正变换的结果，A1A_1A1​ 则是对后半部分做正变换的结果。由此，可以在 O(nlog⁡n)O(n\log n)O(nlogn) 的时间复杂度下解决。等式右边表示将两个多项式直接拼接。

其逆变换如下：

a=(a0,a1−a0)a = (a_0, a_1 - a_0) a=(a0​,a1​−a0​)

其中，a0a_0a0​ 表示 AAA 的前半部分做逆变换，a1a_1a1​ 同理。

按位与

与按位或的正变换类似，但快速进行时基于的性质改变为：

A=(A0+A1,A1)a=(a0−a1,a1)A = (A_0 + A_1, A_1)\\ a = (a_0 - a_1, a_1) A=(A0​+A1​,A1​)a=(a0​−a1​,a1​)

按位异或

首先，定义运算 ∘\circ∘ 表示 iii 和 jjj 做按位与运算后 111 的奇偶性，做如下正变换：

Ai=∑i∘j=0aj−∑i∘j=1ajA_i = \sum_{i\circ j = 0} a_j - \sum_{i \circ j = 1} a_j Ai​=i∘j=0∑​aj​−i∘j=1∑​aj​

基于如下性质，快速做正变换：

A=(Ao+A1,A0−A1)A = (A_o + A_1, A_0-A_1) A=(Ao​+A1​,A0​−A1​)

a=(a0+a12,a0−a12)a = (\dfrac{a_0 + a_1}{2}, \dfrac{a_0-a_1}{2}) a=(2a0​+a1​​,2a0​−a1​​)
