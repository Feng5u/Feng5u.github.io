[25赛季暑期集训 0812] D - Kiritorisen

Statement

给定一个长度为 nnn 的字符串，计算有多少个三元组 (l,r,d)(l,r,d)(l,r,d)，满足 s[l,r],s[l,d],s[d+1,r]s[l,r],s[l,d], s[d + 1, r]s[l,r],s[l,d],s[d+1,r] 都是回文串。

1≤n≤3×1061 \leq n \leq 3 \times 10^61≤n≤3×106

Tutorial

有以下结论：

  1. 如果一个长度为 nnn 的回文串有一个长度为 kkk 的回文后缀，那么它有一个长度为 n−kn-kn−k 的周期；
  2. 如果一个字符串有两个长度为 d1、d2d_1、d_2d1​、d2​ 的周期，且 d1+d2≤nd_1 + d_2 \leq nd1​+d2​≤n，那么它有一个长度为 gcd⁡(d1,d2)\gcd(d_1,d_2)gcd(d1​,d2​) 的周期；
  3. 如果一个回文串可以被分为两个回文的前后缀，那么它一定是某个回文串重复大于等于两次形成的，且前后缀的长度都是该回文串的倍数；
  4. 如果一个回文串有一个长度为 ddd 的周期，且 d∣nd | nd∣n，那么它的最小周期也可以整除 nnn

通过以上结论，我们可以知道：对于一个回文串，如果它的最长回文后缀长度为 ddd，自身长度为 lll。那么它有一个最小周期，长度为 n−dn-dn−d，如果 (n−d)∣n(n-d) | n(n−d)∣n，那么该最小周期重复了 nn−d\dfrac{n}{n-d}n−dn​ 次，且该最小周期也是一个回文串，因此对于当前回文串 (l,r)(l,r)(l,r)，有 nn−d−1\dfrac{n}{n-d}-1n−dn​−1 个 xxx 可以被选择，满足 (l,r,x)(l,r,x)(l,r,x) 是一个合法的三元组。

通过回文自动机，我们可以处理出来每个本质不同的回文串以及它的最长回文后缀，最后答案就是每一个本质不同的回文串的 nn−d−1\dfrac{n}{n-d} - 1n−dn​−1 乘以该回文串的出现次数的和。
